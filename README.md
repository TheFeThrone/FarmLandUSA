# Farmland USA

A game from 2000 by ValuSoft in association with KO Interactive Inc. and CANOPY GAMES
## Installation

1. Setup
    - Launch Setup.exe
2. Patch
    - Launch Patch.exe
      - Fix the path
3. Fix
    - Copy everything in this Folder to the installation path
    - Run fix.bat
